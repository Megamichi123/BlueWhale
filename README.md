- Windows 10 & 11 or +

- Latest version of <a href="https://www.python.org/downloads/">Python</a>.

1 - Download the .zip folder.
2 - Unzip the folder.
3 - Launch "setup.py".
```
Or
```
1 - Open a terminal.
2 - > git clone ...

4 - > git pull
5 - > python3 setup.py
```
